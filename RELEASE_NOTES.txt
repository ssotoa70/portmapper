Q: ModuleNotFoundError: No module named 'PyQt6'
A: Install dependencies:
   python3 -m pip install PyQt6 Pillow pandas requests --break-system-packages

   Or use the launcher:
   ./run.sh

Q: "python3: command not found"
A: Python 3 is not installed. Install it using:
   - macOS: brew install python3
   - Ubuntu/Debian: sudo apt install python3
   - Windows: Download from python.org or use WSL

Q: Application won't start
A: Check Python version (needs 3.9+):
   python3 --version

   Verify all dependencies:
   python3 -m pip list | grep -E 'PyQt6|Pillow|pandas|requests'

Q: "Permission denied" when running run.sh
A: Make the script executable:
   chmod +x /Users/sergio.soto/portmapper/run.sh

Q: Projects not saving in expected location
A: Ensure the portmapper folder has write permissions:
   ls -ld /Users/sergio.soto/portmapper  # Check permissions
   chmod 755 /Users/sergio.soto/portmapper  # Fix if needed
